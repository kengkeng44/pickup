📌 規則總結 (違反 = revert)

  1. 3-section 強制: top / middle / bottom 必須清楚分離
  2. 頂 = chrome 框架 sticky fixed 不滾; 中 = scrollable 可滾; 底 = nav / CTA fixed
  3. 所有按鈕一律 SquishButton (見 pickup-rn CONVENTIONS.md)
  4. 不要對 fixed element 固定元素 加 CSS transition CSS 過渡 (iOS Safari jitter 抖動)
  5. chrome 顏色一律 cream #f1ebe1 跟 viewport bg 背景 一致
  6. Haptic 觸感 Medium 配按下, Success 配答對
  7. TTS 文字轉語音 auto-speak on Q mount + cleanup 清除 on unmount
  8. MC 選擇題 不要 question prompt 問題提示, 選項自說明 (B.280)
  9. 答錯 2-strike 兩次錯誤: 1st 紅 retry, 2nd 知道了 reveal 揭曉答案
  10. 書封 dynamic chapter 動態章節 = IntersectionObserver 視窗偵測器, 不用 scrollY listener 滾動事件監聽
