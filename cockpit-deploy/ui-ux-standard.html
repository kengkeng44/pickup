🎨 Pickup UI/UX 標準 v1

User directive 2026-06-10:「每一個頁面從上到下分三個區塊」
規則: 任何 page / screen 必須 lock 死 3-section vertical structure: TOP chrome / sticky · MIDDLE scrollable content · BOTTOM nav 或 CTA fixed. 改之前必須對齊此 spec.

📌 規則總結 (違反 = revert)

  1. 3-section 強制: top / middle / bottom 必須清楚分離
  2. TOP = chrome sticky fixed 不 scroll; MIDDLE = scrollable; BOTTOM = nav 或 CTA fixed
  3. 所有按鈕一律 SquishButton (見 pickup-rn/CONVENTIONS.md)
  4. 不要對 fixed element 加 CSS transition (iOS Safari jitter)
  5. chrome cream #f1ebe1 跟 viewport bg 一致
  6. Haptic Medium 配 press, Haptic Success 配 correct
  7. TTS auto-speak on Q mount + cleanup on unmount
  8. MC 不要 question prompt, 選項自說明 (B.280)
  9. 答錯 2-strike: 1st 紅 retry, 2nd 知道了 reveal
  10. 書封 dynamic chapter = IntersectionObserver, 不用 scrollY listener
