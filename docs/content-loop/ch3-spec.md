# Ch3 拾光 (Pickup) Content Spec — 醜小鴨 (The Ugly Duckling)

> **Status:** v2.0 content spec draft  •  **Author beat:** Mochi 在奶奶院子聽奶奶說醜小鴨  •  **Target:** A2 Taiwanese learners  •  **Lessons:** 25  •  **Questions:** ~152 (6 Q standard)

---

## 1. Story Arc (~250 字)

Mochi 第三晚回到奶奶院子。秋風起,矮牆有點涼,Hana 已經在草地上等。奶奶從藤椅旁拿起那本舊故事書,翻到一張畫著池塘的頁面 —— 今晚是醜小鴨。

故事從鴨媽媽孵蛋開始。五顆白蛋一一裂開,可是最後一顆又大又灰,鴨媽媽不解,但仍把那隻最大的小鴨抱在翅膀下。下了水,大家驚訝:這隻笨手笨腳的「醜小鴨」其實會游泳。可是兄弟姊妹笑他、咬他,連雞舍裡的母雞和老貓都嫌他長得不對。最後連鴨媽媽都嘆氣:「孩子,你走遠一點吧。」

醜小鴨離開家,獨自走過沼澤、躲過獵人的槍聲、被一戶農家短暫收留又被孩子追趕。秋天落葉,他看到一群白色長頸大鳥飛過,心裡一震 —— 那是什麼?他說不出名字,只覺得想跟上去。

冬天最冷的那段日子,他凍在冰湖邊,差點死掉。一位農夫把他撿回家救活。春天來了,他飛到湖邊低頭喝水,看見水裡的倒影 —— 是一隻白天鵝。原來他不是醜,他只是還不知道自己是誰。一群天鵝靠過來,把他當成同伴。

奶奶闔上書。Mochi 想:我也常被叫流浪貓,可是這個院子告訴我,我是「{catName}」。

**核心情感:** 自我認同的延後揭曉。不是「努力變美」的勵志線,是「你一直都是這個樣子,只是周圍人看錯了」。
