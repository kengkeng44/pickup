### A.5 共通 theme 分析 (2020-2026 跨四獎)

**高頻 theme cluster**:
1. **文化身份 / BIPOC / 移民經驗** — Newbery 7/7 年都有 (New Kid 黑人 / When You Trap a Tiger 韓裔 / Last Cuentista 拉丁裔 / Freewater 黑奴 / Watercress 華裔 / Chooch Cherokee). 也是 Caldecott 主軸.
2. **家庭 / 手足 / 跨世代** — 祖母 (Tiger), 兄妹 (Chooch), 母女記憶 (Watercress).
3. **環境 / 自然 / 動物視角** — Eyes & the Impossible (狗), Water Protectors (水), Hot Dog (都市熱浪).
4. **失落 / 哀悼 / 韌性** — Blues in Sky, Undefeated, Big.
5. **小語種 / 民間故事重述** — Cherokee, Korean folklore, Indigenous Australia.

**Pickup 啟示**: 純白人主角故事 / 純歐美設定已非獎項主流. ELT 內容若想對齊「現代兒童文學氣質」, 應該:
- 含多元文化角色 (亞裔 / 黑人 / 原住民)
- 含家庭情感主題 (不只是「冒險」)
- 用動物 / 自然當載體比用「冒險打怪」更得獎

## C. 中華市場熱門 IP + 為什麼

### C.1 台灣現役暢銷童書 (誠品 / 博客來 / 親子天下交叉確認)

| 書名 / 系列 | 作者 / 原產 | 累計銷量 (台灣) | 為什麼紅 |
|------------|-----------|----------------|---------|
| 神奇柑仔店 (1-18 集) | 廣嶋玲子 / 日本 | 全系列 100 萬冊+ | 「兒童版解憂雜貨店」, 道德兩難 + 奇幻 + 短篇可單獨閱讀 |
| 屁屁偵探 | Troll 監修 / 日本 | 國小圖書館排行常客 | 推理 + 屎尿屁幽默, 男童殺手 |
| 麵包小偷 | 柴田啓子 / 日本 | 日本 550 萬冊, 台灣熱賣 | 角色設計 + 反差萌 + 暖心 |
| 怪傑佐羅力 | 原裕 / 日本 | 系列長賣 30 年 | 反派主角 + 失敗也 OK 主題 |
| 短耳兔 | 達文茜 / 唐唐 / 台灣 | 暖心繪本指標 | 接納自己, 台灣本土原創 |
| 充滿祕密的魔石館 | 廣嶋玲子 / 日本 | 神奇柑仔店姊妹作 | 同作者奇幻短篇 |
| 小學生最實用的生物事典 | 日本翻譯 | 知識類常勝 | 圖鑑風潮 |

### C.2 信誼幼兒文學獎 (台灣最具份量幼兒繪本獎, 2024 擴大為「信誼兒童文學獎」)

- 第 32-37 屆 (2019-2024) 持續推出原創幼兒圖畫書
- 主題以**幼兒生活經驗** (吃飯 / 上廁所 / 睡覺 / 情緒) 為主
- 2024 起新增「兒童文學組」, 接受橋樑書/中篇

### C.3 中國大陸暢銷童書趨勢 (2024-2025)

- **「comic+」風潮** — 知識漫畫 / 學科漫畫成長最快
- **經典童話常青** — 安徒生 / 格林 / 伊索 / 一千零一夜 四大套書是家庭標配
- **2024 諾貝爾文學獎 (韓江) 帶動兒童書連帶銷量** (Publishing Perspectives 2025/12 報告)

### C.4 香港市場特性

- 香港**沒有官方暢銷追蹤系統** (OpenBook 不含香港)
- 主要通路: 商務 / 三聯 / 中華 / 誠品香港 / Bookazine
- 童書市場小但**雙語消費力強** (英 + 繁中並行)

### C.5 中華市場為什麼買單這些?

1. **道德 / 因果主題** — 神奇柑仔店每集都是「貪心會出事 / 善行有回報」, 完全對齊華人教養 frame
2. **可單篇閱讀** — 屁屁偵探 / 怪傑佐羅力都是 1 本 = 1 完整故事, 適合每天 1 篇睡前
3. **幽默無底線但無暴力** — 屁屁/麵包小偷 walks the line between cute & cheeky
4. **日本來源強勢** — 7 大暢銷有 5 個來自日本, 翻譯成本低 + 美學接受度高
5. **本土原創偏暖心** — 短耳兔 / 信誼得獎作品偏「療癒」, 不衝突日系冒險路線

### D.6 為什麼 PD 故事跨 50 年都熱?

1. **無版權成本** — 出版社/平台可自由改編, 競爭推動品質
2. **道德骨架簡單** — 一個 lesson per story, 適合 5-10 歲
3. **角色設定極簡** — 龜/兔/狼/公主/王子, 文化遷移無阻
4. **無敘事門檻** — 父母自己小時候就讀過, 願意買給小孩

## E. Pickup 下一輪推薦 ship 排序 (按 4 軸 score)

### E.1 評分方法

每個 IP / 故事題材按 4 軸打分 (各軸 0-5 分):
- **軸 1 (現代獎項對齊)** — 是否符合 2020-2026 得獎主題 cluster
- **軸 2 (全球銷量基數)** — 累計銷售 / IP 影響力
- **軸 3 (中華市場接受度)** — 台灣家長熟悉 + 願意買單
- **軸 4 (PD 可用性)** — 是否可直接改編無版權風險
- **Pickup ELT 適配** — 詞彙難度 / 句型可重述 / 文化中性 / A2 適配

### E.2 推薦 Tier S (滿分, 立刻 ship)

| 排名 | 題材 | 軸 1 | 軸 2 | 軸 3 | 軸 4 | ELT 適配 | 總分 |
|-----|------|------|------|------|------|----------|------|
| 1 | **The Tortoise and the Hare (龜兔賽跑)** | 3 | 5 | 5 | 5 | 5 | 23 |
| 2 | **The Ugly Duckling (醜小鴨)** | 4 (身份認同) | 5 | 5 | 5 | 5 | 24 |
| 3 | **The Boy Who Cried Wolf (狼來了)** | 3 | 5 | 5 | 5 | 5 | 23 |
| 4 | **Goldilocks and the Three Bears** | 3 | 5 | 4 | 5 | 5 | 22 |
| 5 | **The Three Little Pigs** | 3 | 5 | 5 | 5 | 5 | 23 |

**理由**: 5 大故事是 PD + 全球辨識度 + 中華熟悉度都頂格, 改編成 ELT 課程沒版權風險, A2 詞彙可控.

### E.3 Tier A (1-3 個月內 ship)

| 排名 | 題材 | 理由 |
|-----|------|------|
| 6 | The Little Red Riding Hood | PD + 警惕主題 + 對話多 (適合聽力題) |
| 7 | The Ant and the Grasshopper | PD + 季節 vocab + 勤奮主題對華人 frame |
| 8 | Cinderella | PD + 全球辨識 (但需小心 Disney 商標, 用 Grimm 文本基底) |
| 9 | Hansel and Gretel | PD + 食物詞彙密集 (gingerbread / candy / forest) |
| 10 | The Lion and the Mouse | PD + 大小對比 + 互助主題 |
| 11 | The Emperor's New Clothes | PD + 誠實主題 + 中華課本標配 |
| 12 | The Little Match Girl | PD + 冬季 vocab (但情緒沉重, 考慮 A2 兒童適不適合) |

### E.4 Tier B (3-6 個月內 ship, 需研究授權)

| 題材 | 注意事項 |
|-----|---------|
| The Very Hungry Caterpillar 概念 | Eric Carle 文本受保護, 但**毛蟲→蝴蝶 + 食物日常**這個 motif 是非版權的, 可自製類似結構 |
| Peter Rabbit 概念 | Beatrix Potter 部分作品 2023 起進入美國 PD, 但**插圖風格**仍受保護, 文本可用 |
| Pinocchio | Collodi 原文 PD, 但 Disney 版形象不可碰 |
| Wizard of Oz | Baum 原文 PD (1900), Disney/MGM 形象不可碰 |
| Alice in Wonderland | Carroll PD, 但 A2 句法太難, 需大量改寫 |

### E.5 Tier C (避免或長期觀察)

- **Harry Potter / Wimpy Kid / Dog Man / Magic Tree House** — 全部現役版權, 改編需高額授權, 不適合 Pickup 階段
- **Mo Willems 系列** — 版權強勢 + Paramount 動畫即將推出, 不要碰
- **神奇柑仔店 / 屁屁偵探 / 麵包小偷** — 日本現役版權, 改編成本高
- **Disney 童話形象** (Cinderella / Snow White / Little Mermaid 的 Disney 視覺) — 用 PD 原文 + 自製插畫可, 不要用 Disney 美學

### E.6 Pickup 排序執行建議

**第一波 (本季 ship)**:
1. 龜兔賽跑 → speed / slow / fast / win 詞彙群
2. 醜小鴨 → ugly / beautiful / different / family 詞彙群 (對齊現代 BIPOC/身份主題)
3. 狼來了 → lie / truth / shout / help 詞彙群
4. 三隻小豬 → house / brick / straw / wolf 詞彙群
5. 小紅帽 → forest / grandma / basket / wolf 詞彙群

**第二波 (下季 ship)**:
6. 螞蟻和蚱蜢 → 季節 vocab + work/play
7. 灰姑娘 → family / shoe / dance / kind
8. 國王的新衣 → king / clothes / honest

**創新方向 (參考獎項 trends)**:
- 加入**亞裔/原住民/黑人主角**重述 (對齊 Watercress / Chooch / New Kid 主題)
- 加入**動物視角故事** (對齊 Eyes & the Impossible / Hot Dog)
- 加入**家庭情感** (而非單純冒險打怪) — 對齊 When You Trap a Tiger / Last Cuentista

**Pickup ship 排序 final summary**: 龜兔賽跑 → 醜小鴨 → 狼來了 → 三隻小豬 → 小紅帽 5 個 PD 故事為第一波. 全球銷量榜的 Eric Carle / Beatrix Potter / Mo Willems 不直接 ship (版權), 但概念可借鏡 (毛蟲變蝴蝶結構 / 動物擬人 / 早期讀者對話格式). 現代獎項 trend 提示: 下一季加入**多元文化主角**重述經典 (例如 Cherokee 版三隻小豬 / 華裔版灰姑娘) 對齊 Caldecott / Newbery 2020-2026 主流方向, 同時讓 ELT 內容看起來 culturally aware 而非單純復古.
