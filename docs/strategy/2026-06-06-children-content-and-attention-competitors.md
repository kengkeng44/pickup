# Pickup 兒童內容類型 + 注意力競品 研究 (8-12 歲)

> 研究日期: 2026-06-06
> 目的: 為 Pickup (拾光) 內容策略擴張 (題目 / 故事類型) 提供 2 軸定錨
> 軸 1 = 8-12 兒童當前最愛的內容類型 (不只童話)
> 軸 2 = 每天消耗兒童 2-4 小時的「注意力競品」+ Pickup 可學 / 不可學機制
> 引用全部附 URL, 未來可追溯

---

## TL;DR (給趕時間的 stakeholder)

1. **童話只佔 1/6 — Pickup 7 童話已經是「老人視角」的內容布局**: 2026 年 8-12 兒童最熱的內容是 (a) **Roleplay 社交** (Brookhaven / Adopt Me / Toca World), (b) **養成 / Idle / Tycoon** (Grow A Garden / Steal a Brainrot), (c) **動物 + 家庭日常** (Bluey / Pocoyo), (d) **神話 + Isekai 異世界** (Percy Jackson / Reincarnated as a Slime / Dreamslinger), (e) **科學 + 探索懸疑** (神奇柑仔店 / 屁屁偵探 / 怪傑佐羅力), (f) **創造 / 建造 沙盒** (Minecraft / Roblox Studio). 童話只是 (d) 的子集.

2. **Pickup 該加 3 大類型, 砍 1 類型**: 加 → 動物 + 家庭日常 (Bluey 風) / 神話異世界 (Percy Jackson 風) / 科學探索懸疑 (屁屁偵探風). 砍 → 過度說教 / 公主王子 / 完美結局童話 (8-12 已不吃).

3. **Pickup 該抄 5 招上癮機制**: (1) Duolingo streak + freeze (-21% churn), (2) Roblox 化的「自選 narrative」角色, (3) Adopt Me 化的「**收集 + trade**」(把單字當寵物收集), (4) Brookhaven 化的「**aspiration** = 扮演英雄」, (5) 每章 reveal 後解鎖 **collectible card** (神奇寶貝圖鑑機制).

4. **Pickup 該避 3 招**: (a) Steal a Brainrot 的 **pay-to-win + 玩家互搶** (違反兒童 ELT 安全), (b) Cocomelon 的 **無情節 stim loop** (違反 narrative comprehension 學習), (c) Roblox 的 **無監管社交** (家長拒絕主因, 違反 Pickup 雙語安全定位).

5. **預測 (bull / base / bear)**:
   - **Bull**: Pickup 加「神話異世界 + 收集機制」, 6 個月內 retention D30 從 ~8% → ~18% (Duolingo 級別)
   - **Base**: 內容多元化幫助 D30 從 ~8% → ~12%, 但 collection / streak 機制疊加複雜度 = ship 慢
   - **Bear**: 拉太多新類型 → 失焦, 雙語負擔倍增 + QA 成本爆, D30 反而掉到 ~5%

## A. 兒童內容類型 6 大類

下面 6 類各給 5 個 example IP + 為什麼 8-12 愛 + 共通 theme/hook + Pickup 採納度.

### A1. 動物 + 家庭日常 (Slice-of-life Animal Family) ⭐⭐⭐⭐⭐ 強烈建議採納

**為什麼這年齡層愛**:
8-12 兒童進入「同理心成熟期」 — 開始把動物擬人化當「比自己更小」的弟妹照顧, 同時透過動物角色處理現實家庭情緒 (吵架 / 分離 / 嫉妒) 而不需要直面. Bluey 之所以 2025 年單一節目 **45 billion minutes** 觀看, 因為孩子和家長**同時**找到投射對象.

**Example IP** (5 個):
1. **Bluey** — 2025 美國串流冠軍, 45 billion minutes watched, Disney+ 鎮店
2. **Pocoyo** — 50 channels / 22 languages, 月 26M unique viewers, 已與 Lingokids 合作
3. **Peppa Pig** — Disney+ 經典 IP, 8-12 兒童雖過了核心年齡但弟妹輩仍養大這代
4. **Sago Mini** (Sago Mini World) — 早教 + 動物可愛風, 與 Toca Boca 共 15M+ MAU
5. **Cocomelon** — 200M subscribers (YouTube 兒童冠軍), 雖在 8-12 已過齡但 IP recognition 100%

**共通 theme / hook (對應 Pickup B1-B6 hook 框架)**:
- B1 (相對 = 比較) → 動物家庭 vs 人類家庭的反差
- B2 (對比 = 反差) → 大小體型 / 性格差異 (Bluey vs Bingo)
- B5 (溫情 = warmth) → 「家人在」結尾 (Bluey 每集尾聲)
- B6 (好奇 = curiosity) → 動物視角看人類社會 (Peppa Pig)

**Pickup 該不該採納**: ✅ **強烈建議加入**
- Pickup 目前已有 cat/dog mascot, 升級成「Mochi 一家」(媽媽 + 小貓 + 小狗) 立刻有 Bluey 級情緒共鳴
- ELT 適配度滿分: 家庭場景 = A2 最高頻 vocabulary (eat / sleep / play / mom / dad / hungry / tired)
- 雙語負擔最低: 家庭情緒詞彙中文 1:1 對應乾淨, 翻譯 placeholder substitution 不會亂

---

### A2. 神話 + Isekai 異世界 (Mythology + Isekai) ⭐⭐⭐⭐⭐ 強烈建議採納

**為什麼這年齡層愛**:
8-12 進入「身分認同覺醒期」 — 喜歡「我其實是被選中的人」narrative. Percy Jackson 第二季 2026 年回歸, Isekai 在 anime 圈被定義為「all-age 但特別吃 8-14」, 因為**異世界 = 重新洗牌的公平起點**, 對校園焦慮的孩子是逃生口.

**Example IP** (5 個):
1. **Percy Jackson and the Olympians** (Disney+ 2024 起, 9-12 最強推薦) — 希臘神話 + 校園
2. **Dreamslinger** (Graci Kim, 2025 中文圈翻譯熱) — 韓國神話 + 夢境
3. **Vanya and the Wild Hunt** (Sangu Mandanna, 2025 SLJ Best Book) — 世界民間傳說 + 神經多樣性主角
4. **How to Free a Jinn** (Raidah Shah Idil, 2025) — 馬來西亞 jinn 神話
5. **Reincarnated as a Slime** (anime 8-14 高度推薦) — Isekai 始祖之一

**共通 theme / hook**:
- B3 (因果 = consequence) → 神話原因論 (為什麼有四季 / 為什麼下雨)
- B4 (轉折 = twist) → 主角的真實身分 (Percy 發現自己是半神)
- B6 (好奇) → 異世界規則探索 (魔法系統 / 怪物圖鑑)

**Pickup 該不該採納**: ✅ **強烈建議加入**
- Pickup 7 童話 + 此類「神話進階」是天然順序 — 童話 = 入門, 神話 = 升級
- ELT 適配度高: 神話故事可天然引入 sequence (first / then / finally) + cause-effect (because / so) — A2 → B1 銜接
- 文化平衡: 別只西希臘神話, 加東亞 (中國 / 日本 / 馬來) 神話, 對台灣家長吸引力高
- **具體建議**: 開「奧林帕斯篇」(希臘) + 「妖怪退治篇」(日本) + 「西遊外傳」(中) 三章, 每章 5-7 集

---

### A3. 科學探索懸疑 (Science Mystery / Detective) ⭐⭐⭐⭐ 建議採納

**為什麼這年齡層愛**:
8-12 是「邏輯推理萌芽期」 — 開始享受「我比角色先猜到答案」的優越感. 台灣金石堂 2025 童書暢銷榜「神奇柑仔店」+「怪傑佐羅力」+「屁屁偵探」三系列**佔前 10 一半**, 反映台灣家長對「閱讀有用」+ 兒童「我會解謎」雙重需求.

**Example IP** (5 個):
1. **神奇柑仔店** (廣島玲子) — 台灣連續 2 年金石堂 / 誠品 top 1-3, 神祕零食 + 後果故事
2. **屁屁偵探** (Troll) — 誠品 2025 TOP100, 解謎 + 幽默, 已動畫化
3. **怪傑佐羅力** (原ゆたか) — 日本圖書館借閱率第一, 搗蛋 + 失敗轉勝
4. **科學實驗王** (大長今) — 韓國科學漫畫, 台灣小學主流必讀
5. **X 尋寶探險隊** — 地理 + 解謎, 國家地理風

**共通 theme / hook**:
- B3 (因果) → 解謎邏輯鏈 (因為 X 所以 Y)
- B4 (轉折) → 兇手 reveal / 反派翻盤
- B6 (好奇) → 科學現象解釋 (為什麼會 / 怎麼辦到)

**Pickup 該不該採納**: ✅ **建議加入** (台灣家長 + 孩子雙叫好)
- ELT 適配度中等: 邏輯詞彙 (because / so / if-then / clue / hint) 對 A2 略難, 需 B1-A2 過渡橋
- 雙語負擔中等: 神祕零食 / 偵探事務所 等專名翻譯需謹慎
- **具體建議**: 模仿「神奇柑仔店」開「神祕單字店」章 — 每集學一個神祕單字, 用了單字後出現意想不到的後果, 5W2H + 因果同時練

### A4. Roleplay 社交 + 自選 narrative (Sandbox Roleplay) ⭐⭐⭐ 機制可抄但內容不直抄

**為什麼這年齡層愛**:
Brookhaven / Adopt Me / Toca World 的核心是「**autonomy = 自主權**」 — 8-12 在現實零控制權 (被告知吃什麼穿什麼去哪), 在遊戲裡選工作 / 開咖啡車 / 住豪宅是**心理代償**. 螢幕中分析「現實版的玩家家」.

**Example IP** (5 個):
1. **Brookhaven RP** (Roblox) — 2026 角色扮演冠軍, 開放城市無目標
2. **Adopt Me!** (Roblox) — 531K concurrent, 寵物收集 + 交易 + 家裝
3. **Toca Boca World** (Toca Boca) — 32 app / 215 國 / 140M+ downloads
4. **Dress to Impress** (Roblox) — 6 分鐘搭配 + 同儕投票, top 3 for 7-12
5. **Sago Mini World** — 早教版 Toca, Spin Master 旗下 15M+ MAU

**共通 theme / hook**:
- 主動 narrative 創造 (玩家自己編故事, 不是被動接收)
- 收集 + 分類 (寵物 / 衣服 / 家具)
- 社交炫耀 (給朋友看自己的家 / 衣服)

**Pickup 該不該採納**: ⚠️ **內容方向不直抄, 但 3 個機制必抄**
- ❌ 不抄: 完全開放沙盒 → ELT 學習目標會散失
- ✅ 抄機制 1: 玩家可選 mascot 性格 / 名字 / 衣服 (Mochi 風格自選)
- ✅ 抄機制 2: 答對解鎖「collectible card」放入「我的圖鑑」(類 Adopt Me 寵物收集)
- ✅ 抄機制 3: 章節之間解鎖「Mochi 的家」場景 — 答對的單字裝飾家 (類 Toca World 家裝)

### A5. 養成 / Idle / Tycoon (Farm + Income Game) ⭐⭐⭐ 機制抄, 不抄主題

**為什麼這年齡層愛**:
**Grow A Garden** 2025 年破 Roblox 同時在線紀錄 (22.3M concurrent), **Steal a Brainrot** 成定義 2025 年的 phenomenon. 共同點: **離線也在長** (idle income) + **看著數字漲** (dopamine loop). 8-12 兒童耐心開始建立, 但需要「不在線也有收穫」的安撫.

**Example IP** (5 個):
1. **Grow A Garden** (Roblox) — 2025 年 22.3M concurrent record, 種菜 idle
2. **Steal a Brainrot** (Roblox) — 2025 phenomenon, meme 角色生產 + 互搶 (爭議)
3. **Stardew Valley** (Steam) — 8+ 推薦, 98% 好評 / 790K 評論
4. **Blox Fruits** (Roblox) — anime 風 RPG, 吃水果獲超能力
5. **Animal Crossing: New Horizons** — Nintendo 家庭主打, 兒童 + 家長共玩

**共通 theme / hook**:
- 離線進度 = 回來看「成長」
- 累積 → 升級 → 解鎖新內容
- 自我表達 (我的農場 / 我的島)

**Pickup 該不該採納**: ⚠️ **機制部分抄, 主題不抄**
- ❌ 不抄主題: 種菜 / 經營 → ELT 適配度差
- ✅ 抄機制 1: 「離開期間 Mochi 仍在學」— 回到 app 看到「Mochi 又學了 5 個單字」(虛擬陪伴)
- ✅ 抄機制 2: 累積式進度條 (vs streak 純消耗) — 即使中斷, 累積學過的單字數字仍在

### A6. 創造 / 建造 沙盒 (Creator Sandbox) ⭐⭐ 不建議直抄

**為什麼這年齡層愛**:
Minecraft 在 8-12 仍是基本盤 (近半數 3-12 兒童玩), Roblox Studio 12+ 兒童開始自己做遊戲. 共同點: **「我的世界, 我來決定」** + 「我做的東西, 別人玩」(炫耀感). Gen Alpha 報告反覆強調**「interactive content wins」**.

**Example IP** (5 個):
1. **Minecraft** — 全平台基本盤, Education Edition 學校採用
2. **Roblox Studio** (creator side) — 12+ 兒童 UGC 起點
3. **LEGO Star Wars: The Skywalker Saga** (Steam) — 9 集電影故事 + 創造
4. **Dreams** (PlayStation) — 全民遊戲創造 (略大齡 12+)
5. **MeepCity** (Roblox) — 早期家裝 + 社交沙盒

**共通 theme / hook**:
- 創造 + 展示 (社交 reward)
- 無 fail state (沒輸, 只有完成度)
- 無限可能 (limit = 想像)

**Pickup 該不該採納**: ❌ **不直抄** (但保留 1 個小機制)
- Pickup 是「英文 → 故事 → 句子完成」, 非沙盒. 強行加沙盒會破壞 ELT 目標
- ✅ 唯一可抄: 「迷你 sentence builder」 — 學完一章後可自己用學過的單字組句子, 句子變成 Mochi 的台詞 (creator + display 雙重 reward)

### B.2 各 app 拆解

#### B.2.1 Roblox — 注意力黑洞之王
- **數據**: Q3 2025 DAU 151.5M, 35% verified under 13 (= ~50M 13 歲以下)
- **上癮機制**:
  - **UGC 無限內容** — 每天有新遊戲, 無 boredom
  - **Robux 經濟** — 兒童學會「我要儲值買 Robux」, 等同進入消費心理訓練場
  - **社交多人** — Brookhaven 整個 server 50 人, 不孤單
  - **roleplay autonomy** — 選工作 / 開車 / 蓋房子 = 心理代償
- **商業模式**: F2P + Robux 微交易, ARPDAU $12.68 bookings (極高)
- **Pickup 該抄**:
  1. ✅ **autonomy** — 讓孩子選自己的 mascot 名字 / 衣服 / 性格 (低成本實作)
  2. ✅ **UGC 元素** — 讓孩子答完課程後「裝飾 Mochi 的家」, 別人 (家長 / 朋友) 來看
- **Pickup 該避**:
  1. ❌ **無監管多人聊天** — 違反兒童 ELT 安全定位
  2. ❌ **儲值經濟** — Pickup 是教育產品, 不是賭場
  3. ❌ **完全 open-ended** — ELT 需 structured progression

#### B.2.2 Cocomelon — 視覺刺激極限案例
- **數據**: 200M YouTube subscribers, 但 Netflix viewership 從 2023 → 2024 跌 60% (Disney+ 接手 2027)
- **上癮機制**: 高飽和色彩 + 重複歌曲 + 0.5 秒切鏡頭 = 嬰幼兒視覺多巴胺 hijack
- **商業模式**: YouTube ad share + 串流授權 (Netflix → Disney+ 2027)
- **Pickup 該抄**: ❌ **完全不抄** — Cocomelon 的成功是 0-3 歲生意, 8-12 學齡兒童反而反感, 且其機制違反 narrative comprehension
- **Pickup 該避**: 高頻切鏡頭 + 無情節 stim loop = 完全違背 Pickup ELT 戰略

#### B.2.3 Bluey — 情緒共鳴典範
- **數據**: 45B minutes watched 2025, 美國串流冠軍
- **上癮機制**:
  - **每集 7-8 分鐘短結構** — 完整起承轉合, 適合短注意力
  - **家長兒童雙向 catch** — 兒童看角色, 家長看自己

## C. Pickup 可抄哪 5-7 招 (具體實作建議)

按「ROI 高 → 低 + 實作難度 低 → 高」排序:

### C.1 ⭐⭐⭐⭐⭐ Streak + Freeze 機制 (Duolingo 直抄)
**為什麼**: ELT 黃金機制, 已被 Duolingo 驗證 — 7 天 streak 3.6x retention, freeze 降 churn 21%.
**怎麼做**:
- 每日完成 1 章 +1 streak day
- 月初送 2 個 freeze (可手動使用 / 自動)
- streak 視覺: 從火苗 → 火焰 → 龍 (像神奇寶貝進化)
- **避**: 不要在 push notification 用 streak 威脅 (對兒童 + 家長有害)
**實作**: 純前端 + 1 個 storage field, 1 週 ship.
**預期影響**: D7 → D14 retention +30-50%

### C.2 ⭐⭐⭐⭐⭐ Collectible Card 圖鑑 (Adopt Me + 神奇寶貝混血)
**為什麼**: 8-12 兒童「收集 + 分類」是核心心理需求 (見 Adopt Me 531K 同時在線). 把單字 / 故事元素變成 collectible card 後**, 學完不會「沒了」, 還在圖鑑裡**, 提升學習價值感.
**怎麼做**:
- 答對解鎖該章節的「故事 card」 (Mochi 釣魚 / Mochi 找錢包 ...)
- 每張 card 顯示: 場景插畫 + 學過的核心句子 + 學過的核心單字
- 圖鑑頁可滑動回顧, 點 card 重播音檔
- 進階: 把 card 分稀有度 (常見 / 稀有 / 傳奇), 完美通關 (3 題全對) 才解鎖傳奇版
**實作**: 中等 — 新 screen + card 視覺 + storage, 約 2 週.
**預期影響**: 故事完成度 (story completion rate) +20-30%

### C.3 ⭐⭐⭐⭐ Mascot Customization (Brookhaven autonomy)
**為什麼**: 8-12 兒童最缺「自主權」, 給 Mochi (cat/dog) 自選名字 / 簡單衣服 / 配件 = 即時 attachment.
**怎麼做**:
- onboarding 第 1 步: 「Mochi 是男生 / 女生? 叫什麼名字? 戴什麼?」
- 之後所有故事 / 對話中用 user 取的名字 (placeholder substitution 機制已有, 見 memory)
- 累積 ticket 解鎖新衣服 / 新配件 (帽子 / 圍巾 / 眼鏡)
**實作**: 中等 — 已有 placeholder substitution, 加 wardrobe UI + storage, 約 1-2 週.
**預期影響**: D1 → D7 retention +15-25% (attachment effect)

### C.4 ⭐⭐⭐⭐ 短結構 + 情緒誠實 narrative (Bluey 風)
**為什麼**: Bluey 45B 分鐘的核心不是動畫品質, 是「情緒誠實 + 短結構」. Mochi 應該會犯錯 / 害怕 / 嫉妒, 不假裝完美.
**怎麼做**:
- 每章 5-7 分鐘故事 + 5-7 題練習 (已對齊現狀)
- Mochi 在故事中**會失敗** (找錢包找不到 / 釣魚釣不到), 然後**和 user 一起學會新單字後**才解決
- 結尾 1 句中文 reveal 帶情緒 ("Mochi 鬆了一口氣" / "Mochi 還是有點害怕但繼續往前")
**實作**: 內容工作, 不是工程. 重寫現有 7 童話的結構約 2-3 週 (per chapter).
**預期影響**: 完成率 +10-15%, 家長 NPS +20+ (情緒共鳴)

### C.5 ⭐⭐⭐ 離線陪伴 (Idle / Tycoon 機制)
**為什麼**: Grow A Garden 22.3M concurrent 證明 idle income 對兒童極具吸引力 — 「我不在, Mochi 也在動」.
**怎麼做**:
- 離線期間 Mochi 會「自己讀繪本」, 每 6 小時學 1-2 個簡單字
- 回 app 看到「Mochi 在你不在的時候學了 'umbrella' 和 'rainbow', 想跟你分享!」
- 小機制, 但**極強情緒 hook**
**實作**: 中等 — 純前端 (last login timestamp + 隨機抽字), 約 1 週.
**預期影響**: 回訪率 (D1 return) +20-30%

### C.6 ⭐⭐⭐ 月主題包 (Toca Boca 月更新)
**為什麼**: Toca Boca 每月新主題包 (學校 / 萬聖節 / 寵物店) 是訂閱 retention 的長效機制. 給家長「訂閱有持續產出」感.
**怎麼做**:
- 每月 1 個新章節 (如 10 月 = 萬聖節 ghost/pumpkin 章, 12 月 = 聖誕 reindeer/gift 章, 2 月 = 過年 lucky/red envelope 章 — 加雙語節慶優勢)

**實作**: 內容 + 內容 produc 流程化, 中長期投資.
**預期影響**: 訂閱 D30 retention +15-25%, 流失率降低

### C.7 ⭐⭐ 迷你 sentence builder (Minecraft 創造機制簡化版)
**為什麼**: Creator content 是 Gen Alpha 的「炫耀感」核心, 但完全沙盒違反 ELT. 折衷: 學完一章可組 3-5 句, 變成 Mochi 台詞分享給家長.
**怎麼做**:
- 章末解鎖「Mochi 想說的話」mini game
- 給 5-8 個學過的單字, 拖拉組句, 句子變成 Mochi 的語音台詞
- 可截圖 / 短影片分享 (家長看到孩子的成果)
**實作**: 中等 — 新 mini game + sharing, 約 2 週.
**預期影響**: 家長分享 (organic 拉新) +50% (假設原本基數低)

## D. Pickup 應避哪 3-5 招 (違反兒童 ELT / 違反 pivot 戰略)

### D.1 ❌ Steal a Brainrot 式 pay-to-win + 玩家互搶
**為什麼**: Steal a Brainrot 雖然是 2025 phenomenon, 但 viral TikTok 影片顯示**「兒童被搶後痛哭」是常態**. 違反兒童 ELT 的安全 + 心理健康定位. Pickup 的家長畫像 (台灣中產家長) 看到一次就刪 app.
**Pickup 戰略**: 絕對禁止「兒童互搶 / 兒童被剝奪」機制. 所有獎勵只能累積不能損失.

### D.2 ❌ Cocomelon 式無情節 stim loop
**為什麼**: Cocomelon 對 0-3 是視覺多巴胺 hijack, 對 8-12 反而反感. 且其機制違反 narrative comprehension (Pickup ELT 核心目標). 高頻切鏡頭 + 純歌曲重複 = 學會聽 melody 但學不會 sentence.
**Pickup 戰略**: 視覺切換間隔 ≥ 3 秒, 句子完整呈現, 不要拼貼影片風.

### D.3 ❌ Roblox 式無監管多人聊天
**為什麼**: Pickup 雙語 ELT 定位 = 學習 + 安全並重. 多人聊天 = 對家長最大紅旗. Roblox 2026 年才被迫做 age verification, 已造成大量家長禁用.
**Pickup 戰略**: 0 chat, 最多異步「家長端看孩子分享」(C.7), 沒有兒童間直接溝通.

### D.4 ❌ Duolingo 式 heart 扣心 + 強制付費
**為什麼**: heart 機制對成人語言學習者 OK (有自驅), 對 8-12 兒童造成挫敗 + 家長反感. Pickup memory 已記載「答錯第 2 次 reveal」(blindRetry 2-strike reveal) — 這是對的方向, 別倒退.
**Pickup 戰略**: 答錯 reveal 即正解, 不扣血. 訂閱用「內容存量 + 月主題包」鎖, 不用「機制 friction」鎖.

### D.5 ❌ Duolingo 式無情 push notification
**為什麼**: Duolingo 那隻綠色貓頭鷹的 push 對成人是 meme, 對兒童 + 家長是 stress. 「你的 streak 即將斷! 快回來!」對 8 歲兒童 = 焦慮. 對家長 = 「這 app 在勒索我兒子, 刪了」.
**Pickup 戰略**: Push notification 對家長, 不對兒童. 訊息語氣是「Mochi 想念你」(柔和), 不是「streak 將斷」(威脅). 每天最多 1 條, 用戶可關.

## E. 對 Pickup 內容策略的最終 3 條建議

### E.1 (最重要) 內容類型: 7 童話 → 「Mochi 宇宙 30 章」3 軸擴張

**新內容矩陣**:
- **童話篇** (現有 7) — 入門
- **Mochi 一家篇** (動物家庭日常, 7-10 章) — A1 風格, 情緒共鳴主力
- **奧林帕斯 + 妖怪 + 西遊篇** (神話異世界, 5-7 章) — A2 風格, 8-12 拉齡
- **神祕單字店篇** (科學懸疑, 5-7 章) — A3 風格, 台灣家長共鳴

**為什麼這順序**:
1. Mochi 一家篇先做 = 利用已有 mascot, 內容雙語負擔最低
2. 神話篇 = 升級感最強, 對 8-12 最具吸引力
3. 神祕單字店 = 台灣家長口碑機制 (神奇柑仔店, 屁屁偵探效應)

**砍**: 完美結局公主王子童話 — 8-12 已不吃.

**預期 ROI**: 內容多樣性 + 雙語家庭認同度 → D30 retention +30-50%

### E.2 (最簡單先做) 上癮機制: 4 個一起 ship — Streak + Freeze + Collectible Card + Mascot Customization

這 4 個機制**互相加乘**:
- streak + freeze → 每日回訪
- collectible card → 完成度 + 收集慾
- mascot customization → autonomy + attachment

實作順序:
1. **Week 1-2**: Mascot Customization (onboarding 改寫, placeholder 已有)
2. **Week 3-4**: Streak + Freeze (純 storage + UI)
3. **Week 5-6**: Collectible Card 圖鑑 (新 screen)

ship 後 2 週做一波 A/B test (帶這 4 機制 vs 不帶), 預期 D7 retention 從 ~25% → ~38%.

### E.3 (避雷) 商業模式 lock: $9.99/月 Family Plan + 月主題包持續產出, 不走 IAP / heart 路

**抄**: Duolingo Family $9.99/月 (6 人) + Lingokids $74.99/年 (60% off 年付) + Toca Boca 月主題包

**避**:
- 不走 ABC Mouse $14.99/月 (高價, 台灣家長價格抵抗)
- 不走 Roblox / Toca Boca IAP pack (Pickup 是教育, 不是娃娃屋)
- 不走 Duolingo heart 強制付費 (兒童不適合)

**Pickup 訂閱 + 月主題包**: 訂閱解鎖所有現有內容 + 月主題包 (萬聖 / 聖誕 / 過年) 額外驚喜.

### 預測 (bull / base / bear)

- **Bull**: Mochi 宇宙 30 章 + 4 機制都 ship, 6 個月後 D30 retention 18-22% (Duolingo 級), 訂閱 conversion 6-8%, 變現可持續
- **Base**: 30 章 ship 但 4 機制只做 2 個, 內容拉新但 retention 卡 12-15%, conversion 3-5%
- **Bear**: 內容拉太多新類型導致雙語 QA 爆 + ship 慢, 機制只做 streak 一個, D30 卡 8% 不動, conversion 1-2% 失血
