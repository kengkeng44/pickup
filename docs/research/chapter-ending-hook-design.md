## A. 文學理論 4 sources

### A1. Bell 《Plot & Structure》HIP + LOCK

每 scene 必須有 Hook / Intensity / **Prompt** (=結尾鉤)。

**Prompt 5 種典型**:
1. 突然揭露的秘密
2. 神秘對話
3. 重大決定
4. 懸而未答的問題
5. 迫近的危險 ← Bell 認為最好用

**LOCK**: Lead / Objective / Confrontation / **Knock-out ending**。每 scene 結尾若沒讓主角「比開場更糟」(stewing over new complication),整個 story 是 marking time。

### A2. Sol Stein — Architectural Suspense

**核心**: 章節結尾**不能解決衝突,必須打開更深衝突** (must not resolve, open up further)。

Cycles of suspense: 同時養多條懸念線,讓讀者在 Ch N 還等 Ch N-2 答案時,Ch N 又開新懸念。

> "The writer's duty is to set up something that cries for a resolution and then to act irresponsibly, to dance away from the reader's problem...prolonging and exacerbating the reader's desperate need for resolution."

### A3. Robert McKee 《Story》— Scene Turning

**Scene turning**: 每 scene 主角的 "value" (正/負電荷) 必須翻轉。正開 → 必負收;反之亦然。

**Beat = exchange of action/reaction**。Beat 不是「事件」,是角色情緒在 expectation vs result 之間的 gap。

**Major gap**: scene 內最強翻轉點 = 「結尾切點」最佳位置。

**對 Pickup 啟示**: 現用 story beat 切法犯錯 — 把「事件序列」當「情緒翻轉」。**切點不該放在「事件結束」,該放在「情緒翻轉發生的瞬間」**。

### A4. Brewer & Lichtenstein 1982 + Zeigarnik 1927 — 認知心理學

**Structural-Affect Theory**: 同組事件,**重排順序**誘發 3 種情緒:
- **Suspense**: 延後結果 (知 stake, 不知 outcome)
- **Curiosity**: 先給結果, 後揭過程
- **Surprise**: 藏關鍵 event 到最後

**Zeigarnik 效應 (1927)**: 未完成任務記憶存留時間 > 已完成。Cliffhanger 直接利用 — 「斷在主角推門前」記憶留存比「斷在主角推門後關門」高約 40%。

**Cambridge 2020s**: Narrative suspense 取決於「potentially inquiry-terminating questions」— 結尾必須留下「答了會結束、答不出會持續」的問題。

## B. 6 種 Hook Framework

### B1 物理懸念 (Literal Cliffhanger)
主角身陷不可逆物理險境,章末停在「動作中途」。
- Hunger Games Ch5: "everything seems frozen in time. Then the apples spill to the ground and I'm blown backward into the air."
- Da Vinci Code 視角輪替放大物理懸念

### B2 情緒翻轉 (Emotional Twist)
McKee value charge 翻轉 — 內在 state 在最後一句倒反。
- Misery: Annie 從 nurse → captor 在同一個微笑下
- Gone Girl Part 1 結尾: Amy 日記轉視角揭露她是 narrator
- 1984 Part 2 結尾: O'Brien 揭身分 "You are the dead."

### B3 資訊缺口 (Information Gap)
給線索 + 主角反應,但不給內容。
- Harry Potter Ch1: Dumbledore 放嬰兒於門口 → 4 個未答問題啟動全書
- Hunger Games Reaping: "And it's not me." — 半截信息

### B4 期待加速 (Anticipation)
主角剛跨入新場景,門剛開鏡頭就切。
- Da Vinci Code: Langdon 推開密室門 → 章末
- Hobbit Ch1: Bilbo 簽約答應冒險

### B5 道德兩難 (Moral Dilemma)
章末停在「選 A 還 B」的當下。
- Sophie's Choice 整本書 = 一個選擇,每次回憶在「她張嘴前」剪斷
- Crime and Punishment 1 部末: Raskolnikov 在房門外手持斧頭,動作未做

### B6 預言種子 (Foreshadowing)
章末用一個「象徵物 / 環境變化」暗示風暴將至。
- ASOIAF Prologue 末: "Winter is coming."
- Macbeth Act 1: "Fair is foul, and foul is fair"

## D. 設計原則 — 每切點問 3 題

1. **(McKee)** 這切點 value 翻轉了嗎? 還是只「事件結束」?
2. **(Stein)** 這切點 open up 新衝突,還是 resolve 舊衝突?
3. **(Brewer/Cambridge)** 留下 inquiry-terminating question 嗎? 能在一句話內 phrase「我想知道___」?

3 題若有 1 題答「否」,切點往前/後挪 1-2 句。

## F. narrative-cut-analyst skill prompt template

```
Role: narrative-cut-analyst (CEFR-A2 listening pedagogy aware)

Input:
  - Story text (segmented by sentence)
  - Target lesson count N
  - Target avg lesson length (seconds of audio)

Steps:
  1. Per-sentence emotion tag (Plutchik 8-point or VAD)
  2. Plot emotional arc (sentence index vs valence)
  3. Identify peak candidates (local maxima of |dValence/dx|)
  4. Per candidate cut, classify hook type B1-B6
  5. Score each candidate:
     - Inquiry-terminating question presence
     - Value charge flip (McKee)
     - Open-not-resolve (Stein)
     - Suspense gradient (Doust & Piwek path-count proxy)
  6. Select top N-1 cuts maximizing total score
  7. Output: cut indices + hook type + "想知道..." per cut
```
