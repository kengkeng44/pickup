## E. Top 3 Actionable (S effort — 30 min 以內)

### 🥇 #1 OptionBtn 加 ✓ / ✗ 圖示 (renderers.tsx:112-127, S=30min)

```tsx
// correct state 左側加綠勾
{state === 'correct' && <span style={{ marginRight: 6 }}>✓</span>}
// wrong state 左側加紅叉
{state === 'wrong' && <span style={{ marginRight: 6 }}>✗</span>}
```

同時加 `aria-label={`${label}${state === 'correct' ? ' — 正確' : state === 'wrong' ? ' — 錯誤' : ''}`}` 讓 screen reader / 色盲 / 靜音用戶三路都能感知反饋。

**ROI**: S(30min) × 高影響(所有用戶的答題反饋,色盲/靜音雙場景)= ⭐⭐⭐

### 🥈 #2 地圖 done 節點加 ✓ checkmark 小 badge (MapPage.tsx:240ish, S=20min)

在已完成節點右上角疊一個小 ✓ icon (12px, white text, 圓形 olive bg),純文字或 CSS,不需要新圖片。

```tsx
{isDone && (
  <span style={{ position: 'absolute', top: -4, right: -4, width: 18, height: 18,
    background: '#7d9a4f', borderRadius: '50%', color: '#fff',
    fontSize: 11, fontWeight: 900, display: 'flex', alignItems: 'center', justifyContent: 'center' }}>✓</span>
)}
```

**ROI**: S(20min) × 中影響(色盲 + 一般用戶進度確認感)= ⭐⭐
