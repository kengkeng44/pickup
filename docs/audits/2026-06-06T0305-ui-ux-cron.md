## A. Angle research (5 bullets)

- **Tap before you read** — every Brilliant module opens with a micro-interaction (drag, tap, slider) before any prose; the interaction *is* the lesson, not a quiz at the end.
- **Concept node labels always visible** — course map shows a 1-2 word concept label beneath every node icon; scannable at a glance; user knows exactly what they're entering before tapping.
- **Progressive scaffolding strip** — within a lesson, current step is highlighted in a mini "step strip" (◎—○—○—○); types differ by shape/color (watch vs interact vs problem).
- **Segment dividers break long sequences** — on a 24-step course, Brilliant inserts named section header cards ("Section 2: Build the Concept") between groups of 4-6 nodes; map never looks like 24 identical blobs.
- **Understated completion** — Brilliant lesson end: single large check + 1 sentence + 1 CTA ("Great work → Continue"). No leaderboard, no confetti storm; calm affirms mastery.

---

## B. Pickup 現狀 (對齊這 angle)

- **Map = 24 identical paw icons** — `storyBeat` field exists in lesson JSON and is used only in `aria-label`; no visible text label under any node. 24 greyed/golden circles with zero scannable content.
- **Narration is fully passive** — `NarrationRenderer` fires `onAdvance` automatically 2 s after TTS ends (renderers.tsx:162); no Continue button is rendered; user can't slow down or speed up. Zero agency between steps.
- **No segment grouping on map** — `segmentType` (`outer-prologue` / `main-story` / `aesop-side` / `outer-outro` / `review`) is in lesson data but never rendered as a divider; 24 nodes scroll as one undifferentiated column.
- **CompletePanel: 5 simultaneous stimuli** — 12 confetti spans + 🎉 emoji + triptych XP/ACCURACY/TIME + hook inquiry dashed-border card + implicit back button all appear at once; conflicts with "understated completion" best practice for 8-12 children.
- **q-counter is content-agnostic** — `q3/14` chip tells position but not type; user can't distinguish narration chunk from hard listen-mc; no sense of "scaffolding ramp" within lesson.

- File: `src/react-app/pages/MapPage.tsx` — inside node `<button>` JSX
- Add below the `<img>` icon:
  ```tsx

    <span style={{
      position: 'absolute', bottom: -18,
      left: '50%', transform: 'translateX(-50%)',
      fontSize: 9, fontWeight: 700, color: COLOR_TEXT_MUTED,
  ```

- Why: 24 blind paw blobs → 24 named goals; scan → tap conversion for children.

- File: `src/react-app/renderers.tsx` — `NarrationRenderer`
- Add `const [showContinue, setShowContinue] = useState(false)` state.

- Why: children need agency + acknowledgment between story beats; auto-advance removes learning dwell time.
