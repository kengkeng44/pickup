## A. Angle research — Duolingo Streak/League patterns

- **Streak flame is the hero metric**: 🔥N is displayed top-center on every screen, updates in real time, and is treated as the user's identity metric — not XP or level.
- **Streak Freeze shield**: players can spend 10 gems (or get 1 free/week) to pre-arm a shield that absorbs one missed day. The shield is visually overlaid on the flame icon. Studies cite -21% 30-day churn when Streak Freeze is available.
- **Streak milestones trigger celebration moments**: at 7 / 14 / 30 / 60 / 100 / 365 days a full-screen toast with confetti + badge unlock fires. These are the app's peak emotional moments — users share screenshots on social.
- **Weekly XP goal (not league)**: below the streak, Duolingo shows a weekly XP bar (50/100/250 XP target). Finishing a lesson visibly fills the bar — immediate re-engagement hook for casual users who skip leagues.
- **Streak wager** (gem mechanic): Duolingo lets users "bet" gems that they'll maintain the streak this week. Loss-aversion anchors commitment. Converts passive streak-holders into active streak-defenders.

## B. Pickup 現狀 — 對齊 Streak/League angle

- **Two streak systems exist and are disconnected**: `data/streak.ts` writes daily streak to localStorage, but `MapPage`/`TasksPage`/`ProfilePage` all display `useRunStore(s => s.streak)` — which is in-session answer-streak (resets to 0 on every page load). **The real daily streak is stored but never shown.**
- **`updateStreak()` is only called in Phaser `BootScene.ts`**: the React lesson complete path (`LessonPage` CompletePanel) never calls `updateStreak()`, so completing a lesson in the React app doesn't advance the daily counter.
- **AlertsPage achievements are permanently greyed out**: `AlertsPage.tsx` uses a static hardcoded array with `earned: false` for all 8 badges. The dynamic `evaluateAchievements()` function in `data/achievements.ts` (which reads real XP, streak, chapter progress) is never imported into the React layer. 3-day and 7-day streak badges can never unlock.
- **No Streak Freeze mechanic**: master matrix ROI #2 (churn -21%), 5-person-day estimate — not yet implemented.
- **No weekly XP goal bar**: no re-engagement structure below the streak. Users who aren't motivated by raw streak count have no short-term goal.
- **No streak milestone celebration**: 7-day / 30-day streaks trigger nothing — the peak emotional moment is silently skipped.
