## A. Angle research (5 bullet)

- **Content-first entry**: Cake opens with a 5-second real media clip — user is already consuming before any account or preference form appears. "Watch first, sign up later" delays friction until after value is felt.
- **Immersive Watch→Shadow→Check loop**: Three-step micro-lesson (1. watch clip, 2. shadow-repeat with waveform visual, 3. fill-in-blank check) fits in <60s. Users exit first session knowing it works.
- **No blank-slate first screen**: Cake shows a recommended clip on launch, not a menu or map. The algorithm — even for cold-start — shows trending/popular content. Zero choice paralysis.
- **Single focal point per screen**: Each screen is one thing: one video, one sentence, one tap. No sidebars, no grids, no multi-chapter arrays.
- **Hook sentence ≤ 8 words**: Clip titles and spotlight sentences are short enough to read at a glance on 5.4" screen while thumb-scrolling. Recall: Pickup A2 target is exactly this range.

---

## B. Pickup 現狀 (對齊這 angle)

- **No 5-second hook**: First screen = MapPage with 31-chapter infinite scroll. Cold-start user sees a wall of paw-icon nodes with no story context, no preview of what they will learn. Cake's pattern demands immediate content taste before navigation.
- **SplashView / LevelTest / OnboardingPicker are dead code**: All three components exist (`src/react-app/views/SplashView.tsx`, `components/LevelTest.tsx`, `components/OnboardingPicker.tsx`) but NONE are rendered in `App.tsx` or any page. Zero users see them. This is the biggest cold-start gap.
- **GrandmaRecommendCarousel hidden at xp=0**: `MapPage.tsx` has `GrandmaRecommendCarousel` which hides itself on cold-start (`forceShow` not set). The very users who need orientation most get a bare map.
- **Chapter count overwhelms first-time child**: 31 chapters visible in aggregate mode. Cake surfaces exactly 1 clip. Pickup surfaces potentially 700+ nodes (31×24) in infinite-scroll mode. Cognitive load at first open is extreme for a 9-year-old.
- **No "taste" micro-lesson before commitment**: Cake's hook is the first clip — you shadow-repeat before you know you're in a lesson. Pickup requires: open app → scroll map → tap node → load JSON → lesson page → first question. That's 5 steps before value delivery.

## D. Top 3 actionable (S effort)

### #1 Wire LevelTest gate (P0 — S 30min)
**檔案**: `src/react-app/pages/MapPage.tsx`
**改法**: 在 component 頂部加：
```tsx
const [levelDone, setLevelDone] = useState(
  !!localStorage.getItem('pickup.ability.level')
);
if (!levelDone) return <LevelTest onComplete={() => setLevelDone(true)} />;
```
LevelTest 已有完整 UI + write 邏輯，只差這一個 gate。
**ROI**: 全新用戶體驗從「一堆節點沒方向」→「5題診斷 → 有能力標籤 → 地圖」。
