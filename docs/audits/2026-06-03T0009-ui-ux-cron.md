# Pickup UI/UX Cron — 2026-06-03 00:09 UTC

Today's angle: **Drops 5-min session pacing**
Source: https://languagedrops.com / App Store product page + UX teardowns (re: timed-session design, garden metaphor, per-word time estimates)

---

## A. Drops pattern (5 bullets)

- **Hard 5-minute countdown**: Drops shows a circular progress ring that counts down from 5:00. Session ends hard when the timer hits 0 — scarcity creates focus. Users can pay to remove the cap but free tier trains the "5 min = enough" habit.
- **Per-word-set time estimate**: Each topic card in the word library shows "~4 min" before you tap in, so users can judge whether they have time. Zero cognitive friction for commuters ("I have 3 stops, can I finish this?").
- **Session complete = Garden watered**: After 5 minutes a "garden watered today" visual fires. Not "lesson complete" — the metaphor is maintenance, not achievement. Emotional register = calm > anxious.
- **Midpoint pulse**: At ~50% completion a subtle ring pulse shows "halfway there" without breaking flow. Keeps users oriented in the remaining time window.
- **Daily commitment framing**: Onboarding asks "how many minutes per day?" → default 5 min. All streak/badge logic anchors to that commitment, not to "lesson" units.

---

## B. Pickup 現狀 (Drops lens)

- **No per-lesson time estimate on map nodes**: Ch1 has 24 nodes ranging 5-15 Q each. A user cannot tell whether tapping a node costs 2 min or 8 min. Biggest friction for the "下班逃逸" 10-minute window demographic.
- **Session time displayed only in CompletePanel**: `timeStr` shows elapsed time after the lesson ends — too late. Drops shows time *before* the user commits.
- **Daily goal = "完成 1 lesson" not "5 minutes"**: TasksPage anchors to lesson-unit completion. Lessons vary 5-15 Q; time varies 1.5-6 min. A user who does a 15-Q lesson gets the same check as a 5-Q lesson, but the experience is 3× longer.
- **No midpoint indicator inside LessonPage**: Progress is shown only as `q{idx+1}/{total}` — a fraction number with no spatial / temporal framing. No halfway celebration or orientation cue.
- **Narration auto-advance is slow for commuters**: `fallbackMs = max(7000, words × 600 + 4000)` — a 12-word narration sentence blocks ~11 seconds. In a 5-min Drops session that single card would eat 4% of the total time budget silently.

1. **Per-node `~N min` time badge** — `MapPage.tsx` node render: add `<span style={{fontSize:10,color:'#fff',opacity:0.8,marginTop:2}}>{Math.max(1,Math.round(l.questions.length*25/60))}m</span>` below paw icon. Zero backend needed; derived from `l.questions.length` already in scope.
