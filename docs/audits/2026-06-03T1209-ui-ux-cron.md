## A. Angle research (5 bullet)

- **Headline metric = "Known Words" count.** LingQ makes word count the primary progress signal — shown in the dashboard, profile, course page. Watching it tick up is the core dopamine loop.
- **Every word has a mastery tier.** New words: yellow. Recognized: blue. Familiar: light green. Known: no highlight. Reading a lesson = encountering colored words and progressively clearing them.
- **Word-level SRS, not sentence-level.** You save a "LingQ" (flashcard) per individual word, not per whole sentence. SRS re-surfaces that word in varied contexts until it turns white.
- **Per-lesson "new words" preview.** Before opening a lesson, LingQ shows "X blue words" (new vocab) and "Y LingQs" (words needing review). Sets expectation; makes each lesson feel like a collection quest.
- **WordHint / inline tooltip** is universal — tap any word anywhere in the app for definition. But tapping also **logs the encounter**, adding it to the word's history. Lookup = implicit learning signal.

Add helper at bottom:

export function logWordSeen(word: string): void {
  try {
    const raw = localStorage.getItem('pickup.words.seen') ?? '{}';
    const map: Record<string, number> = JSON.parse(raw);
    map[word] = (map[word] ?? 0) + 1;
    localStorage.setItem('pickup.words.seen', JSON.stringify(map));
  } catch {}
}
export function readWordCount(): number {
  try {
    const raw = localStorage.getItem('pickup.words.seen');
    if (!raw) return 0;
    return Object.keys(JSON.parse(raw)).length;
  } catch { return 0; }
}
```

**`src/react-app/pages/ProfilePage.tsx`** — import `readWordCount` + add 5th Stat grid cell:

```
