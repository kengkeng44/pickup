# Pickup Icon Prompts — Duolingo-Style

> 給 ChatGPT / Gemini / Midjourney 用。每個 prompt 都是「完整 self-contained」可以直接複製貼上。

---

## 🎯 風格鎖(全部 icon 共用,先記著)

```
Style for ALL icons:
- Flat vector illustration, NO 3D, NO realism
- Bold solid colors, vibrant but slightly muted (Pickup palette)
- Pixel-clean simple geometry
- NO outline OR very thin (1px) outline
- Square 1:1 aspect ratio, 1024x1024
- Transparent background (PNG with alpha)
- Centered subject, ~70% of canvas
- Subtle drop shadow underneath
- Inspired by Duolingo's icon design system (Lin, Junior characters)
- NO text, NO logo, NO frame, NO watermark
```
